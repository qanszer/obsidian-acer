# CCF Live Prod Roles Checklist

This is a checklist I can use to see all the preparations to be done, as well as the reminders for during live

## Main Graphics Operator

**PREPARATION**

1. **Lyrics Section**
	- Check that all songs are present
	- Check if there are no lining errors; if present, use reflow document first
	- Check CCLI and add as necessary
	- Recheck the arrangements for each song
2. **Announcements Section**
	- Login on Chrome using the Live Prod account
	- Go to Gdrive bookmark < Announcements; Reorder list to 'last modified' and download all files after Sunday last week
	- Download the sunday flow and script files
	- Refer to the sunday flow file (2nd page) for the announcements' order
3. **Script Section**
	- Refer to the script file for the addition of the script to the stage displays
	- If there is no toggle option for the stage display on the left video view, force quit the app and toggle stage display on the app's preferences

**ACTUAL/LIVE**

- Start the loop 1 hour before service
- After starting the loop, set the countdown timer in advance
- Start the countdown verses 10 minutes before service
- Show the timer message after the videos in countdown verses end
