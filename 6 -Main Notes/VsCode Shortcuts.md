# VsCode Useful Shortcuts


Add cursor above/below:
ctrl + shift + uparrow/downarrow

Move line up/down:
alt + uparrow/downarrow

Toggle side bar:
ctrl + b

Create new file:
ctrl + alt + n

Move to another file:
ctrl + pg up/down

Close file:
ctrl + w

Delete line:
ctrl + shift + k
