# 1st Semester

## Prelim


### 02/02/26 - Intro to UI/UIX

CX > UX > UI

#### UI - User Interface
- Visual Design
- Motion Design
- Interface Design

UI is the visual layout and interactive elements---how the system looks and operates.

UI is what the user sees.

**Elements:**
- Layout/grids
- Buttons, inputs, icons
- Typography
- Color schemes
- Navigation menus
- Animations & micro-interactions

**UI Principles**
- Consistency
- Visual hierarchy
- Clarity
- Affordance
- Feedback

**Deliverables**
- Style guides
- Component libraries
- High fidelity mockups
- Design systems


**Step by Step Process**

1. Setting design goals
2. Conducting user research
3. Creating a wireframe
4. Developing the visual design
5. Prototype
6. Handling over to developers
7. Gathering user feedback



#### UX - User Experience
- Usability
- Accessibility
- Desirability
- Interaction Design
- Information Architecture
- User Research
- Content Strategy
- Prototyping

UX is the overall feeling and usability of a product. It is what the user feels.


**Step by Step Process**

1. Problem/problem
2. Roadmap to get started
3. Research
4. Analyze
5. Design
6. Validation
7. Handover

#### CX - Customer Experience
- Experience Design
- Brand Reputation
- Sales Process
- Marketing
- Customer Service
- Customer Journey
- Product Delivery
- Store/Outlet/Office
